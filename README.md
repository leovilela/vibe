# vibe
Instruções pra executar o projeto.

Instalar o Node.js (https://nodejs.org/en/).

Baixar o projeto e entrar na pasta "consultas".

Na linha de comando executar o comando 'npm i' para instalar as dependências.

Executar o comando 'npm run serve:mock' para executar o servidor com os dados mockados.

Executar o comando 'npm run dev' para executar o projeto.
